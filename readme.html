pyPrediktorUtilities¶

Helper functions for Prediktor projects

The included classes and functions allows for easier and more maintainable code when working with Prediktor projects. The main features are:

  • Sending emails

  • Working with FTP and SFTP server

  • Logging

  • Generating files through templates
